# Dr. P Post-Purchase Map

Static deployment for the Dr. P post-purchase flow visual map.

## Vercel

Deploy this repo on Vercel as a static site:

1. Import `prtzprayzxc/dr-p-post-purchase-map` into Vercel.
2. Keep the default project settings.
3. Deploy.

The site root `/` rewrites to `/visuals/drp-all-flows-simple.html`.
